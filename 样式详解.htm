帮我标出显示区高度，及输入框高度用注释标出来 初始版本设置为  70vh （视口高度的70%） 根据之前的对话需求修改为  35vh  最终优化版调整为  40vh （当前值）   /* 调整下方数值即可 */ max-height: 40vh; /* 可改为 50vh、60vh 或固定像素值如 500px */   相关高度参数说明：  max-height ：控制显示区的最大展开高度  min-height ：保证折叠状态下仍有120px的可视区域  overflow-y: auto ：内容超过高度时显示滚动条  40vh  表示视口高度的40%（1vh = 视口高度的1%）   /* 输入框高度设置 */ #inputText { width: 100%; height: 140px; /* 输入框高度 */ padding: 12px; border: 2px solid #e5e7eb; border-radius: 8px; margin: 12px 0; font-size: 15px; resize: vertical; } /* 标题样式 */ h1 { color: var(--deepseek-blue); text-align: center; font-size: 22px; /* 标题字体大小 */ margin: -5px 0 15px 0; /* 标题边距（上 -5px，右 0，下 15px，左 0） */ }   关键点说明： 输入框高度由  height: 140px  设置 标题尺寸由  font-size: 22px  定义 标题边距使用简写格式： 上边距 -5px（负值用于微调位置） 下边距 15px 左右边距 0（通过简写格式的第二个和第四个值体现）
